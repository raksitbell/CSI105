Create a Node with OOP

A Node is the smallest building block. It holds one piece of data (value) and a pointer to the next node (next). When we first create a node, next is always null because it's not connected to anything yet.

class Node {
    constructor(value) {

Build the LinkedList Class

The LinkedList class manages a chain of nodes. It tracks the starting point (head), the total count (size), and gives the list a name and ID so we can manage multiple lists. 

// From script.js â€” The LinkedList class
class LinkedList {
    constructor(id, name) {
        this.id   = id || Date.now().toString();
        this.name = name || "New List";
        this.head = null;   // Points to the first node
        this.size = 0;      // How many nodes in this list
    }
}
ðŸ’¡ When a list is created, head is null and size is 0 â€” an empty list.

Push â€” Add to the End

push() adds a new node at the end of the list. If the list is empty, the new node becomes the head. Otherwise, we walk through the chain until we find the last node, then attach the new one. 

// From script.js â€” push method
push(val) {
    const node = new Node(val);
    if (!this.head) this.head = node;
    else {
        let current = this.head;
        while (current.next) current = current.next;
        current.next = node;
    }
    this.size++;
}

Unshift â€” Add to the Front

unshift() adds a new node at the front. The new node's next points to the old head, then the new node becomes the new head. This is faster than push because we don't need to walk the chain. 

// From script.js â€” unshift method
unshift(val) {
    const node = new Node(val);
    node.next = this.head;
    this.head = node;
    this.size++;
}

Get â€” Find a Value by Index

get(index) walks through the list step by step, counting until it reaches the right position. If the index is out of bounds (negative or too large), it returns null. 

// From script.js â€” get method
get(index) {
    if (index < 0 || index >= this.size) return null;
    let current = this.head;
    for (let i = 0; i < index; i++) current = current.next;
    return current.value;
}
