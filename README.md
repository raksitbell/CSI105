# CSI105T - Data Structures Workshop

An interactive collection of data structure and algorithm visualizations built to aid students in understanding the core logic behind standard operational routines. No complex backend or installation logic needed, simply access directly via your browser.

## Workshops Included

1. **[Management Linkedlist](Management%20Linkedlist/)**
   - Manage and visualize singly linked lists with interactive operations. Explore pushing, popping, shifting, unshifting and node mapping logic explicitly.

2. **[Stack & Queue](Stack%20and%20Queue/)**
   - Compare Last-In-First-Out (LIFO) and First-In-First-Out (FIFO) mechanics visually. Includes simultaneous push/pop vs enqueue/dequeue arrays side-by-side.

3. **[Infix Conversion](Infix%20Conversion/)**
   - Built to handle Shunting Yard algorithms translating Infix mathematical expressions to Prefix and Postfix variants dynamically.

4. **[Quick Sort](Quick%20Sort/)**
   - Visualizes parallel divide-and-conquer mechanics traversing Fixed Pivot (Ascending) models sequentially frame-by-frame. Tracks pivot selections, bounds testing logic, and round splits.

## Running Locally

Because all workshops leverage raw client-side logic mapped by HTML & JavaScript directly manipulated in the DOM, deploying it takes just a browser tab:

1. Clone or download the repository.
2. Open the main `index.html` natively within any preferred browser (e.g. Chrome, Edge, Firefox).
3. Navigate between workshops using the homepage central interface links.

## Technologies Used

* **Design**: Tailwind CSS (CDN deployed globally for simple footprint management) & custom CSS.
* **Typeface**: `Kanit` sourced from Google Fonts.
* **Logic Framework**: Vanilla JavaScript utilizing state frames or direct DOM node mutations depending on workshop configurations.
* **Architecture**: Fully frontend compatible layout scaling from desktop to mobile without explicit backend routing mechanics.
