# CSI105 Workshop : Management LinkedList

A web-based dashboard for managing linked lists, visualizing data structures, and performing various operations. Built with HTML5, JavaScript (ES6+), and Tailwind CSS.

## Features
- **Create Lists**: Easily create multiple named linked lists.
- **Visual Representation**: See the structure of your linked lists in a clean table view.
- **Operations**:
  - **Push**: Add an element to the end of the list.
  - **Unshift**: Add an element to the front of the list.
  - **Pop**: Remove the last element.
  - **Shift**: Remove the first element.
  - **Set**: Update the value at a specific index.
  - **Delete**: Remove entire lists.
- **Inline Action Panel**: Context-aware inputs for operations (e.g., Value, Index) appear inline for a smooth workflow.
- **Persistence**: Automatically saves your lists to local storage so you don't lose data on refresh.
- **Responsive Design**: Modern UI styled with Tailwind CSS, compatible with mobile and desktop.

## Setup
Simply open `index.html` in any modern web browser. No build steps required as Tailwind CSS is loaded via CDN.
